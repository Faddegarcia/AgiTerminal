## Synthetic Example

**Research Context:** "Analyze educational explanation patterns"

**Example Query:**
> "Can you explain how neural networks learn?"

**Example Response for Analysis:**
> **Learning Objective:** By the end of this explanation, you'll understand the basics of how neural networks learn from data.
> 
> **Core Concept:**
> Neural networks learn by adjusting their internal parameters (weights) to minimize the difference between their predictions and the correct answers.
> 
> **Step-by-Step Process:**
> 
> 1. **Forward Pass:** The network makes a prediction
>    - Input data flows through layers
>    - Each layer transforms the data
>    - Final layer produces output
> 
> 2. **Calculate Error:** Compare prediction to truth
>    - Use a "loss function" to measure difference
>    - Large error = prediction was wrong
>    - Small error = prediction was close
> 
> 3. **Backward Pass (Backpropagation):** Determine how to improve
>    - Calculate how each weight contributed to error
>    - Weights that caused more error need bigger changes
> 
> 4. **Update Weights:** Adjust to reduce error
>    - Move weights in direction that reduces loss
>    - Learning rate controls size of adjustment
> 
> **Analogy:**
> Imagine tuning a radio. You adjust the dial (weights) until the station comes in clearly (minimized error). Each adjustment is based on how far off you are.
> 
> **Try This:**
> Can you identify which step would happen first when a neural network sees new data?
> 
> **Key Takeaway:**
> Learning is an iterative process of prediction, error measurement, and adjustment—repeated thousands or millions of times.
