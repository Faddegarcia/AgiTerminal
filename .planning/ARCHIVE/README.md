# Planning Archive

This directory contains planning documents from earlier iterations of the project.

## Evolution

### Phase 1: AI System Prompt Library (Archived)

**Date:** Early 2026
**Concept:** A curated repository of AI system prompts from major providers

**Original Goals:**
- Collect system prompts from 5+ providers
- Organize by model/provider
- Document extraction methodology
- Provide educational resources

**Status:** 
The System Prompt Library concept has evolved into the broader **AgiTerminal** educational research platform. The original planning documents are preserved here for reference.

### Phase 2: AgiTerminal Research Framework (Current)

**Date:** February 2026
**Concept:** Comprehensive educational platform for AI system prompt research and benchmarking

**Expanded Goals:**
- System prompt collection (maintained from Phase 1)
- 5-level testing framework for educational purposes
- Python package for programmatic analysis
- Jupyter notebooks for hands-on learning
- Provider comparison tools
- Synthetic examples library
- CLI tools for researchers
